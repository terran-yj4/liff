### 概要
LIFFアプリで対応しているの全てのメッセージを送信するためのものです。
LIFFのURLにパラメータを加えることで自在にメッセージ送信を可能にします。<br>
対応しているメッセージタイプは以下
- テキストメッセージ
- スタンプメッセージ
- 画像メッセージ
- 動画メッセージ
- 音声メッセージ
- 位置情報メッセージ
- テンプレートメッセージ
- Flexメッセージ

詳しくは以下参照
https://developers.line.biz/ja/reference/liff/#message



## 使い方
送信したいトークを開いたら、LIFFにアクセス。
https://liff.line.me/1656316634-09bXLKaV
タイプや詳細情報を入力したら、送信可能。

#### パラメータを付ける場合
例) https://liff.line.me/1656316634-09bXLKaV?type=txt&txt=ok
